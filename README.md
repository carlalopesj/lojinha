# Lojinha - Angular

Esse projeto foi realizado com o propósito de certificação de um projeto de extensão realizado pela minha faculdade. Com ele pude ter experiências com Typescript e Angular. Acho que ter conhecimentos de React, JS e Java me auxiliaram e fizeram com que meu aprendizado fosse um pouco mais rápido, devido algumas semelhanças e conceitos. Pude conhecer mais também sobre componentes, dependências e serviços com o Angular.

O sistema consiste no cadastro de pacientes e produtos, no qual cada paciente da lista possui um carrinho próprio. Foi utilizado o localStorage como armazenamento. Para saber mais sobre esse projeto leia a documentação: [Documentação - Projeto Angular.pdf](https://github.com/user-attachments/files/16582140/Documentacao.-.Projeto.Angular.pdf)

Para testá-lo você pode clonar esse repositório ou fazer o download.zip. Após isso siga os seguintes comandos no terminal:
  1. npm install (instalará as dependências) 
  2. ng serve (rodará o servidor, basta clicar no link gerado do localhost)
